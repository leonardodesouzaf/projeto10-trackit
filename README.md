## Run Locally

Clone the project

```bash
  git clone https://github.com/leonardodesouzaf/projeto10-trackit.git
```

Go to the project directory

```bash
  cd projeto10-trackit
```

Install dependencies

```bash
  npm i
```

Start the server

```bash
  npm run start
```
